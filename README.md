# Linguistic-Probing-Small-BERT

This repository contains the code and datasets used in our bachelor's thesis (KULeuven). 
There is code that we used to to transform Universal Dependencies (UD) data into linguistic probing tasks compatible with the SentEval framework and the datasets (Dutch and Romanian) recieved from them.
There is also the script used for the probing itself.

## References & Paper
This project was developed as part of our  (dutch) research into linguistic probing for small BERT models.

*   **Paper**: Taalkundige Informatie in Kleine Modellen: Een
Meertalige Probing-Analyse van BERT-varianten in
het Engels, Nederlands en Roemeens
*   **Authors**: Jesse Lema & Obe Nowak
